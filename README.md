# Smart AI Email Assistant

An AI-powered email assistant that helps users generate professional and context-aware email replies using Google Gemini API.

## Features
- AI-generated email responses
- Context-aware and professional replies
- Chrome Extension support
- RESTful backend APIs
- Clean and responsive UI

## Tech Stack
- Backend: Spring Boot
- Frontend: ReactJS
- AI: Google Gemini API
- Tools: Git, GitHub
- Extension: Chrome Extension

## Use Case
- Automates email reply writing
- Saves time and improves communication quality

## Future Enhancements
- Email tone customization
- Multi-language support
- User authentication
